  1. Setiap bahasa pemrograman terdapat framework dikarenakan agar supaya aplikasi yang dibuat ada standarnya sehingga tidak seenaknya sendiri membuat aturan. Hal ini penting jika aplikasi yang dibuat melibatkan banyak orang, sehingga semua bisa berkolaborasi dengan baik karena pengkodean yang terstandar dan terstruktur.

    Kegunaan Framework antara lain:

    • Mempermudah dalam membangun aplikasi
    • Menstandarisasi kode program yang di buat
    • Mempercepat dalam membangun aplikasi
    • Memiliki fitur keamanan yang selalu update
    • Dapat menambahkan library atau dependency dengan mudah

  2. Fungsi dan penggunaan if else, while, while do, switch case, for, foreach

    • if else digunakan untuk menjalankan kode program dalam keadaan tertentu. If else efektif jika digunakan untuk menjalankan kode program yang kompleks.

    • while digunakan untuk menjalankan program selama parameternya terpenuhi(true)

    • while do sama dengan while, akan tetapi perbedaanya adalah program berjalan setelah parameternya terpenuhi.

    • switch case digunakan untuk menjalankan program sesuai dengan parameter. switch efektif digunakan apabila parameternya memiliki banyak case dan sederhana.

    • for digunakan untuk mengulang program sesuat dengan parameter yang diberikan.

    • for each digunakan untuk mengiterasi secara satuan dan mengeksekusinya kode program dari sebuah array

  3. Constructor, method, class, object, variabel, parameter, function, procedure merupakan:

    • Constructor merupakan kode program yang dijalankan otomatis ketika sebuah objek dibuat.

    • Methode merupakan sebuah fungsi yang ada di dalam Class

    • Class merupakan rancangan dari sebuah objek.

    • Object merupakan sebuah variabel atau entitas yang dibuat dari sebuah Class.

    • Variabel merupakan sebuah data yang digunakan untuk menyimpan nilai.

    • Parameter merupakan data yang diterima oleh fungsi.

    • Function merupakan kode program yang dapat di eksekusi jika dipanggil.

    • Procedure merupakan kode program yang bersifat prosedural.

  4. Perbedaan method=”Post” dan “Get” adalah

    • Data yang dikirimkan Post tidak terlihat, sedangkan get terlihat di bagian link sebagai query.
    • Methode Post cocok digunakan untuk mengirimkan data yang bersifat sensitif. Sedangkan Get cocok digunakan untuk megnirimkan data yang bersifat umum.

  5. Konsep mengenai Web Service adalah layanan yang digunakan untuk merequest data ke server. Fungsi yang digunakan untuk memanggil web service client adalah dengan menggunakan HTTP request

  6. MVC merupakan paradigma pemrograman yang terdiri dari model, view, controller.

    HMVC merupakan paradigma pemrograman yang terdiri dari model, view, controller akan tetapi bersifat Hybrid, yang mana antara backend dan frontend terpisah.

  7. Fungsi curl() digunakan untuk mengirimkan request ke server.

  8. Sebuah array di PHP ditampilkan menggunakan fungsi print_r() atau var_dump()

  9. Perbedaan HTTP dengan HTTPS adalah tingkat keamananya, HTTPS sudah dilengkapi keamanan melebihi HTTP.

  10. API secara sederhana dapat diartikan sebagai jembatan penghubung antara hardware dengan software atau server dengan client. Contohnya adalah driver hardware, Restful API
